mbedtls.mpi Module API¶

Multi-precision integer library (MPI).

class mbedtls.mpi.MPI¶

Bases: object

Multi-precision integer.

This class implements numbers.Integral. The representation of the MPI is overwritten with random bytes when the MPI is garbage collected.

The bitwise operations are not implemented.
