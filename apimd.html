mbedtls.hash Module API¶

Generic message digest wrapper (hash algorithm).

class mbedtls.hash.Hash¶

Bases: mbedtls._md.MDBase

Wrap and encapsulate hash calculations.

This class is a wrapper for the hash calculations in the md module of mbed TLS. The interface follows the recommendation from PEP 452 for unkeyed hashes.

Parameters:name (bytes) – The MD name known to mbed TLS.
digest_size¶

int – The size of the message digest, in bytes.

block_size¶

int – Not implemented.

name¶

bytes – The name of the message digest.

block_size

The internal block size of the hash algorithm in bytes.

copy()¶

Return a copy (“clone”) of the hash object.

digest()¶

Return the digest output of message.

digest_size

The size of the resulting hash in bytes.

hexdigest()¶

Like digest except the digest is returned as a string object of double length.

name

The canonical name of the hashing algorithm.

class mbedtls.hmac.Hmac¶

Bases: mbedtls._md.MDBase

Wrap and encapsulate HMAC calculations.

This class is a wrapper for the HMAC calculations in the md module of mbed TLS. The interface follows the recommendation from PEP 452 for keyed hashes.

Parameters:
  • key (bytes) – The key to use.
  • name (bytes) – The MD name known to mbed TLS.

Warning

The message is cleared after calculation of the digest. Only call digest() or hexdigest() once per message.

int – The size of the message digest, in bytes.
